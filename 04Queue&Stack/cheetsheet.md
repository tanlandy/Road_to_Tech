# Queue
## Initiate
`Queue<String> q = new LinkedList<>();`
## Methods
```Java
// 队的头尾，在尾部加元素，头部拿走元素
q.offer(2) // O(1)
q.poll() // O(1)
q.peek() // O(1)
q.size() // get size
```

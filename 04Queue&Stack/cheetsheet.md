# Stack
## Initiate
`Stack<String> stack = new Stack<>();`
## Methods
```Java
// 栈顶栈底
stack.push() // O(1) 放入栈顶
stack.pop() // O(1) 删除并且return栈顶
stack.peek() // O(1)

```

# Queue
## Initiate
`Queue<String> q = new LinkedList<>();`
## Methods
```Java
// 队的头尾，在尾部加元素，头部拿走元素
q.offer(2) // O(1)
q.poll() // O(1)
q.peek() // O(1)
q.size() // get size
```
